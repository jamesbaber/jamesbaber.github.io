Project Delv & ChatTarget

ChatTarget • May 1024

One of my first projects was called Project Delv. The project became unmanageable and so was abandoned. A year or so later ChatTarget replaced it. ChatTarget is a free on-line chat client. Visit ChatTarget.co.nf for information. It is fully embeddable in games and other projects, web, desktop and mobile. It is the embedded chat system in 2Dash.

2Dash

Web based mini game • March 2007 - February 2010
